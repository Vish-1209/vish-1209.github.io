1. Customer Churn Prediction for a Bank (Classification)

Objective: Predicted probability of customer churn using structured banking data (10,000 records).

Performed data cleaning, EDA, and feature engineering (e.g., encoding, scaling).

Tools: Pandas, Scikit-learn, XGBoost, SMOTE,Matplotlib

2. Food Delivery Time Prediction (Regression)

Objective: Predicted delivery time using real-world geolocation and order metadata (45,000+ records).

Engineered key features like distance using geodesic calculations and prep time from timestamps.

Handled missing values and inconsistent time formats during preprocessing.

Trained and tuned Linear, Decision Tree, Random Forest, and XGBoost Regressors using GridSearchCV.

Achieved best performance using XGBoost, evaluated with MAE, RMSE, and R².

Saved models with pickle and prepared data for production-like inference pipeline..

Tools: Pandas, Scikit-learn, XGBoost, GridSearchCV, geopy, NumPy
